### **COLORES ACCESIBLES (WCAG)**

#### **COLORES 100% AAA**
Para texto blanco normal (16pt+) sobre fondo de color, y para uso sobre fondo blanco como hiperenlaces:
- **Color AAA Primario**: `#336114`
- **Color AAA Rojo**: `#A20011`
- **Color AAA Morado**: `#53297D`
- **Color AAA Azul**: `#065E84`
- **Color AAA Burdeos**: `#630235`
- **Color AAA Gris**: `#545454`

#### **COLORES 100% AA**
Para texto blanco más grande (18pt+) y en negrita sobre fondo de color:
- **Color AA Primario**: `#44841A`
- **Color Rojo**: `#E70052`
- **Color Rosa**: `#E43989`

#### **COLORES PARCIALMENTE AA**
Para texto blanco más grande (18pt+) y en negrita sobre fondo de color:
- **Color Rojo Oscuro**: `#FF1D34`
- **Color Naranja**: `#F16E22`
- **Color Azul**: `#0B9CDA`

#### **IMPORTANTE**
Utilizar solo con texto negro o por estética, **NO con información escrita**:
- **Color Primario Claro**: `#BECE45`
- **Color Amarillo**: `#FBC43A`
- **Color Fondo Neutro**: `#EAEADE`

## ✍️ Tipografía (Typography)

### **Fuentes Oficiales de la Marca**

[NOMBRE_PROYECTO] utiliza las siguientes tipografías oficiales:

| Fuente                      | Uso Principal                                           | Notas                                                   |
| :-------------------------- | :------------------------------------------------------ | :------------------------------------------------------ |
| **[TIPOGRAFIA_TITULOS_1]**  | Títulos breves y expresivos (≤10 palabras)              | Lado apasionado y expresivo                             |
| **[TIPOGRAFIA_TITULOS_2]**  | Títulos principales, citas, encabezados web             | Tono acreditado y serio, no para cuerpo de texto        |
| **[TIPOGRAFIA_CUERPO]**     | Cuerpo de texto principal (negrita y redonda)           | Fuente condensada, puede perder legibilidad en tamaños pequeños |
| **[TIPOGRAFIA_APOYO]**      | Texto pequeño (≤7pt), web, apps                         | Fuente de ancho completo para legibilidad en espacios reducidos |

---

### **FUENTES PARA TÍTULOS**

#### **Fuente Primaria para Títulos**
**[TIPOGRAFIA_TITULOS_2]**  
Se reserva para titulares, citas y casos de uso visual. Ideal para mensajes con tono acreditado o serio: eslogan, titulares web, etc.

> **Ejemplo:** "[Frase de ejemplo sobre el propósito del proyecto]."

#### **Fuente Secundaria para Títulos**
**[TIPOGRAFIA_TITULOS_1]**  
Solo para enunciados muy breves (≤10 palabras) con tono apasionado y expresivo.

> **Ejemplo:** "[Frase corta y expresiva]."

---

### **FUENTES PARA CUERPO DE TEXTO**

#### **Fuente Primaria para Cuerpo de Texto**
**[TIPOGRAFIA_CUERPO]** (negrita y redonda)  
Fuente condensada para contenidos online e impresos. Pierde legibilidad en tamaños pequeños.

> **Ejemplo:** "[Párrafo de ejemplo del cuerpo de texto]."

#### **Fuente Secundaria para Cuerpo de Texto**
**[TIPOGRAFIA_APOYO]** (negrita y normal)  
Fuente de ancho completo para tamaños pequeños (≤7pt) en sitios web, aplicaciones y situaciones donde el espacio y legibilidad sean críticos.

> **Ejemplo:** "[Texto pequeño de ejemplo para metadatos o notas]."

### **Notas de Implementación**

- Las fuentes **[TIPOGRAFIA_TITULOS_1]**, **[TIPOGRAFIA_CUERPO]** y sus variantes son propias de la marca [PROYECTO].
- **[TIPOGRAFIA_APOYO]** es la fuente secundaria para mejorar legibilidad en textos pequeños y aplicaciones digitales.
- El **[COLOR_PRINCIPAL] ([HEX_PRINCIPAL])** es el color corporativo principal y debe estar presente en todas las comunicaciones.
- La identidad visual de [NOMBRE_PROYECTO] es **digital first**: simple y flexible.
- **Accesibilidad**: Los colores están clasificados según niveles AA y AAA de las Directrices WCAG 1.0.
- Para más información, consultar el **Manual de Marca en Detalle** disponible en la guía oficial.
